# Photography Website

A modern photography website built using **HTML, CSS, and JavaScript**.  
This project showcases a photography studio with different sections including services, about us, and contact information.

## 🌐 Live Website

https://photography-website222.netlify.app/

## 📌 Features

- Responsive photography website design
- Home page with attractive visuals
- About Us section
- Services section
- Contact page
- Image gallery
- Clean and modern UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
